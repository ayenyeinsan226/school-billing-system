# School Management & Billing Platform (SaaS)

A production-ready Full-Stack enterprise application for schools to manage student directories and automate monthly tuition billing system.

## 🚀 Tech Stack
- **Frontend:** React, TypeScript, Tailwind CSS
- **Backend:** Node.js, Express, TypeScript, PostgreSQL

## 🛠️ System Architecture (Monorepo)
- `/frontend` - Rich User Dashboards (Admin, Teacher, Student)
- `/backend` - Scalable RESTful APIs with Clean Layered Architecture

## ✨ Key Features implemented so far:
- [x] Responsive Admin Dashboard UI Layout with Tailwind CSS
- [x] Role-Based UI views (Mock Data setup)
- [ ] Automated Billing Engine via Cron Jobs (Next Phase)
- [ ] Secure JWT Authentication & RBAC Middlewares (Next Phase)
